# 数据库配置指南

## 配置说明

根据您选择的数据库服务，需要在 `application-prod.yml` 中配置相应的连接信息。

## 微信云托管数据库配置

### MySQL配置
```yaml
datasource:
  url: jdbc:mysql://your-mysql-host.tencentcdb.com:3306/valuesfriends
  username: root
  password: @Value1234
```

### Redis配置
```yaml
redis:
  host: your-redis-host.tencentredis.com
  port: 6379
  password: your-redis-password
```

## 腾讯云数据库配置

### MySQL配置
```yaml
datasource:
  url: jdbc:mysql://cdb-xxx.tencentcdb.com:3306/valuesfriends
  username: root
  password: @Value1234
```

### Redis配置
```yaml
redis:
  host: redis-xxx.tencentredis.com
  port: 6379
  password: your-redis-password
```

## 免费数据库配置示例

### PlanetScale MySQL
```yaml
datasource:
  url: jdbc:mysql://aws.connect.psdb.cloud:3306/valuesfriends?sslMode=REQUIRED
  username: your-username
  password: your-password
```

### Redis Cloud
```yaml
redis:
  host: redis-xxx.redislabs.com
  port: 12345
  password: your-redis-password
```

## 配置步骤

1. 获取数据库连接信息
2. 替换 `application-prod.yml` 中的占位符
3. 确保数据库已创建 `valuesfriends` 数据库
4. 运行 SQL 脚本初始化表结构
5. 测试连接

## 注意事项

- 确保数据库允许外网访问（如果部署在云托管外）
- 检查防火墙和安全组设置
- 使用强密码保护数据库安全
- 定期备份重要数据
